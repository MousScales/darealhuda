# Spanish Lessons Translation

This folder contains Spanish translations of the Islamic lessons. The translations follow the same structure as the original lessons but are adapted for Spanish-speaking Muslims.

## File Naming Convention

- Original: `lesson-001.json`
- Spanish: `lesson-001-es.json`

## Translation Guidelines

### Key Terms Translation
- **Allah** - Keep as "Allah" (not "Dios")
- **Quran** - Translate as "Corán"
- **Hadith** - Keep as "hadiz" or "hadices" (plural)
- **Sunnah** - Keep as "Sunnah"
- **Iman** - Keep as "Iman" or translate as "fe"
- **Tawhid** - Keep as "Tawhid" or translate as "Unicidad de Allah"
- **Shirk** - Keep as "shirk" or translate as "asociación"
- **Du'a** - Keep as "du'a" or translate as "súplica"
- **Salah** - Keep as "salah" or translate as "oración"
- **Masjid** - Translate as "mezquita"
- **Ummah** - Translate as "umma" or "comunidad musulmana"

### Formatting
- Maintain the same JSON structure as original files
- Keep Arabic terms in italics when appropriate
- Preserve Quranic references in the format "Corán X:Y"
- Maintain section headings and content organization

### Cultural Considerations
- Use formal "usted" form when addressing Allah
- Maintain respectful tone throughout
- Adapt examples to be culturally relevant for Spanish-speaking Muslims
- Preserve Islamic terminology while making content accessible
